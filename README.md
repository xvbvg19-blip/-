WinUI Annotator (Windows UI 交互标注工具)

1. 软件简介

WinUI Annotator 是一款专为 Windows 桌面 GUI 交互数据采集而设计的标注工具。它可以帮助研究人员或标注员快速在系统截图中定位 UI 元素，并记录其中心点坐标、边界框以及语义功能描述。

核心功能：

 精准定位：支持矩形框标注，并自动计算 UI 元素的几何中心点（Pixel-level）。
 分类标注：内置按钮、图标、输入框、菜单项等多种 Windows 常用 UI 类型。
 交互优化：支持鼠标中键平移图片、滚轮缩放、右键清理队列。
 标准导出：一键生成符合模型训练要求的 JSON 格式数据。

---

 2. 如何运行

本软件已打包为单文件绿色版，无需安装 Python 环境。

1. 启动程序：双击 `WinUI_Annotator.exe` 即可运行。
2. 开始标注：
 点击 “添加文件夹” 导入需要标注的截图。
 在图片上 左键拖拽 绘制标注框。
 在右侧 “UI 类型” 下拉菜单选择类型，并在 “详细描述” 中输入功能描述。
 每一张图标注完成后，按下 Enter (回车键) 自动保存并切换下一张。

3. 如果杀毒软件（360火绒Windows Defender）拦截怎么办？

由于本软件是使用 PyInstaller 打包的未签名脚本，极易被杀毒软件误认为是恶意程序。

解决方法：

 方案 A（推荐）：将软件所在文件夹添加到杀毒软件的 “信任区白名单” 中。
 方案 B：运行过程中若弹出“发现未知程序尝试修改文件”或“拦截行为”，请选择 “允许程序所有操作”。
 原因解释：PyInstaller 打包原理是将 Python 解释器和脚本解压到临时目录运行，这种“自解压并执行”的行为与木马特征相似，但本软件不联网且不修改系统文件，请放心使用。

4. 标注规范提醒 

 中心点原则：坐标点必须落在该 UI 元素的可点击区域内部。
 描述要求：描述应站在用户视角，描述点击后的意图或结果（例如：“Open the Downloads folder in File Explorer”），避免使用 “这是一个按钮” 等无效描述。
 数量要求：根据规范，每张截图建议至少标注 5 个 可交互元素。

5. 快捷键指南

 按键  功能 
 ---  --- 
 鼠标左键拖拽  绘制新的标注框 
 鼠标中键（滚轮按下）  拖动放大后的图片 
 鼠标滚轮  放大缩小图片 
 Delete (Del) 键  删除当前选中的标注框 
 Enter (回车)  保存当前图片标注并跳转至下一张 
 右键点击左侧列表  从队列中移除选中的图片文件

6. 输出格式说明

标注结果将以同名 `.json` 文件形式保存在图片所在目录下，格式如下：

```json
{
  image example.png,
  annotations [
    {
      type 按钮 (Button),
      point [100, 200],   UI 中心点坐标 (x, y)
      bbox [50, 180, 100, 40],  [左上角x, 左上角y, 宽度, 高度]
      desc Close the current window
    }
  ]
}

```
